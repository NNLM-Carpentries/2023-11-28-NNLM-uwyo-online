Python

Python is a popular language for research computing, and great for general-purpose programming as well. We will use Google Colaboratory ("Colab") for this workshop. In Colab, you can make notebooks within which Python code can be written and run. These notebooks will be saved to your Google Drive for you to access and share with others.

If you are familiar with Jupyter notebooks, working with a Colab notebook will feel very similar!

To use Colab, you will need a Google account. This can be an account attached to a Gmail address or another email address.
